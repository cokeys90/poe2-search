# 화면 용어집 · 디자인 가이드

이 앱의 화면 요소를 부르는 **공식 이름**. 대화에서 "그거 좀 고쳐줘"가 어디를 말하는지 서로 헷갈리지 않게 하는 게 목적이다.
새 요소를 만들면 여기에 한 줄 추가한다.

---

## 1. 전체 셸

```
┌──────────┬────────────────────────────────────────────────┐
│          │  상단 바 (헤더)  [파밍 애니]        [테마][⭐]   │
│  사이드바 ├────────────────────────────────────────────────┤
│  (내비    │  본문                                          │
│   레일)   │    ├ 결과 바                                    │
│          │    ├ 필터 카드                                  │
│          │    ├ 서판 종류 바 (서판 탭)                       │
│          │    ├ 콜아웃 (조건부)                             │
│          │    └ 옵션 목록                                  │
│          │                        [즐겨찾기 창] [스크롤 FAB] │
└──────────┴────────────────────────────────────────────────┘
```

| 용어 | 컴포넌트 | 설명 |
|------|----------|------|
| **사이드바** (내비 레일) | `NavRail.jsx` | 좌측 세로 메뉴. 서판 / 경로석 / 정보·문의. 좁은 화면에선 드로어로 열린다 |
| **상단 바** (헤더) | `App.jsx` | 파밍 애니(`FarmingScene`) + 테마 토글 + 즐겨찾기 ⭐ 버튼 |
| **본문** | `App.jsx` `<main>` | 가운데 스크롤 영역 |
| **스크롤 FAB** | `ScrollFab.jsx` | 우하단 떠 있는 점프 버튼(최상단·접두어·접미어) |

## 2. 본문 요소

| 용어 | 컴포넌트 | 설명 |
|------|----------|------|
| **결과 바** | `ResultBar.jsx` | 맨 위 카드. 생성된 **검색어**(정규식)와 글자수(N/250), 가져오기·거래소·복사·초기화, 선택 칩 목록 |
| **검색어** | — | 게임 검색창에 붙여넣는 최종 정규식 문자열. "패턴"과 같은 뜻 |
| **선택 칩** | `ResultBar.jsx` | 결과 바 안, 지금 고른 옵션 하나하나. 포함/제외 뒤집기·최소값·핀·삭제 |
| **필터 카드** | `App.jsx` + `PriceFilter` / `ExtraFilters` | 결과 바 아래 한 장짜리 카드. 왼쪽 열(가격·타락·결합) + 오른쪽 열(등급 격자, 경로석만) |
| **가격 필터** | `PriceFilter.jsx` | 체크박스로 **켜고 끄기**(끄면 입력은 보이되 비활성). 정확히/범위, 값, 화폐 |
| **타락 필터** | `ExtraFilters.jsx` → `CorruptFilter` | 무관 / 타락만 / 비타락만 |
| **등급 격자** | `ExtraFilters.jsx` → `TierGrid` | 경로석 전용. 16T→1T 아이콘 4×4 + "무관" 칩 |
| **결합 모드** | `Segmented.jsx` | OR(아무거나) / AND(모두) |
| **찾기** | `App.jsx` | 옵션 목록 바로 위의 입력창. 목록을 텍스트로 걸러낼 뿐 검색어 생성과는 무관 |
| **서판 종류 바** | `TabletTypeBar.jsx` | 방사능·균열·탐험·환영·의식·감독관·심연·사원 (방사능이 기본·첫 자리) |
| **콜아웃** | `Callout.jsx` | Notion의 info 블록 같은 안내 박스. 내용 길이만큼만 넓어진다 |
| **옵션 목록** | `OptionGroup.jsx` + `OptionRow.jsx` | 그룹별 행 목록. 각 행 = [수치입력 \| 원문(+dev일 때 아래 줄에 거래소명) \| 순서·숨기기]. **배치**: 경로석의 "옵션"(6종)은 전폭, 접두어 = 왼쪽 열 / 접미어(공통·고유) = 오른쪽 열 |
| **dev 배지** | `App.jsx` | 개발 빌드에서만 보이는 UI에 붙이는 초록 배지(`import.meta.env.DEV`). 현재는 "거래소명 보기" 토글 |
| **순서 핸들** | `OptionRow.jsx` | 행 우측 ⇅. 끌어서 순서 변경(즐겨찾기 카드와 같은 방식). 그룹 **안에서만** 이동 |
| **숨기기** | `OptionRow.jsx` | 빨간 ⊖. 누르면 그 그룹의 **숨긴 옵션** 섹션으로 내려간다 |
| **숨긴 옵션** | `OptionGroup.jsx` | 그룹 목록 아래 접힘 섹션. ⊕로 숨기기 해제하면 원래 자리로 돌아온다 |
| **툴팁** | `Tooltip.jsx` | 아이콘 버튼의 힌트. 브라우저 기본 `title`은 커서에 가리므로 **쓰지 않는다** — 대상 버튼 바로 위에 뜨는 이 컴포넌트를 쓴다. 조상에 `overflow-hidden`이 있으면 잘리니 주의 |
| **핀(고정)** | `PinButton.jsx` | 탭을 바꾸거나 초기화해도 살아남는 설정. localStorage `poe2-search:pins` |

## 3-2. 거래소 연동

| 용어 | 컴포넌트 | 설명 |
|------|----------|------|
| **거래소 버튼** | `ResultBar.jsx` | 지금 조건 그대로 공식 거래소를 새 탭으로 연다(`lib/trade.js`의 `tradeUrl`). 거래소로 못 보내는 옵션이 있으면 `-N` 뱃지 |

| **북마클릿** | `lib/trade.js` → `bookmarkletCode` | 거래소 페이지에서 조건을 읽어 앱을 `#trade=…`로 여는 `javascript:` 코드. 링크(검색 ID)로 가져오는 경로는 레이트리밋 때문에 **의도적으로 제거**했다 |

## 5. 색 · 스타일 규칙

Material 3 토큰(`tailwind.config.js`)만 쓴다. 하드코딩 색 금지.

| 쓰임 | 토큰 |
|------|------|
| 배경(층) | `surface-c-lowest` < `surface-c-low` < `surface-c` < `surface-c-high` < `surface-c-highest` |
| 본문 글자 | `on-surface` / 보조 `on-surface-variant` |
| 강조·선택 | `primary`, 선택된 칩·버튼은 `secondary-container` + `on-secondary-container` |
| 테두리 | `outline`(진함) / `outline-variant`(옅음) |
| 위험 | `error` |
| 모서리 | `rounded-md-xs`(4) / `md-s`(8) / `md-m`(12) / `md-l`(16) |

**주의**: 라이트·다크 두 테마에서 모두 읽혀야 한다. `text-primary/60` 같은 반투명 강조색은 배경에 따라 흐려지므로 본문·부가 텍스트에는 `on-surface-variant`를 쓴다.

**z-index 층**: 스크롤 FAB(30) < 즐겨찾기 창(40) < 다이얼로그(50).

**게임 아이콘**: `public/tablet/{slug}.png`(서판 8종), `public/waystone/t{1~16}.png`(경로석 등급). 출처 poe2wiki.net. 렌더는 `GameIcon.jsx`(`TabletIcon` / `WaystoneIcon`)로만 — 밝기 보정이 여기 들어있다.
