# PoE2 경로석·서판 검색기

Path of Exile 2의 경로석(Waystone)·서판(Tablet) 옵션을 게임 내에서 빠르게 찾기 위한
정규식 검색어를 만들어 주는 웹 도구입니다. 원하는 옵션과 수치 범위를 고르면,
게임 검색창에 그대로 붙여넣을 수 있는 검색어가 생성됩니다.

## 사이트

👉 **https://poe2-search.web.app**

## 데이터 출처

게임 옵션 데이터와 헤더 애니메이션의 화폐 이미지(디바인/히네코라의 자물쇠/미러)는
[poe2db.tw/kr](https://poe2db.tw/kr)에서 수집했습니다.
정규식 생성 방식은 오픈소스 [poe2.re (veiset/poe2.re)](https://github.com/veiset/poe2.re)를 참고했습니다.
